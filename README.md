# 团队项目执行看板

当前版本为可离线打开的单文件应用（`index.html`），并且支持**留档/保存**。

## 核心能力
- 首页总览（自动汇总指标）
- 任务清单（可新增/编辑/删除）
- 甘特排期（带日期表头）
- 成员负荷（可新增/编辑/删除成员）

## 你关心的计算规则（已实现）
- 任务人天：不再手填，改为按 `开始日期 ~ 结束日期` 自动计算（含首尾日）
- 成员负荷：按日期维度累计，不会因任务日期重复而“丢失负荷”
  - 计算峰值日负荷（同一天并行任务会叠加）
  - 显示平均饱和度与峰值饱和度
  - 标记超载天数

## 留档与本地数据库能力
- 自动保存：任何任务/成员修改后自动写入浏览器 `localStorage`
- 立即保存：可手动点击“立即保存”
- 快照留档：可创建多版本快照并支持恢复/删除
- 本地数据库文件：支持导出 JSON 文件、导入 JSON 文件（可迁移到另一台机器）

## 打开方式
### 方式 A（推荐）
```bash
python -m http.server 4173
```
访问：`http://127.0.0.1:4173`

### 方式 B
直接双击 `index.html`。

## 说明
- 本页不依赖 npm、不依赖外部 CDN，受限网络也可使用。
- 仓库中仍保留 React + TypeScript + Tailwind 的工程脚手架，便于后续切回工程化开发。
