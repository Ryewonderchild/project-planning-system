# 团队项目执行看板

如果你之前遇到“打不开”，主要有两个常见原因：
1. 依赖 npm 安装（网络受限/403）导致起不来。
2. 页面依赖外部 CDN（公司内网拦截）导致白屏。

## 本版本修复
- `index.html` 改为**纯原生单文件**，不依赖 npm、不依赖任何外部 CDN。
- 支持直接双击打开，或用本地静态服务打开。
- 功能保留：
  - 首页总览
  - 任务清单
  - 甘特排期
  - 成员负荷
  - 自动计算成员饱和度、项目总人天和风险任务数

## 打开方式
### 方式 A（推荐）
```bash
python -m http.server 4173
```
浏览器访问：`http://127.0.0.1:4173`

### 方式 B
直接双击 `index.html`。

如果你希望继续保留 React + TypeScript + Tailwind 的工程化开发流（Vite），可以在网络可用环境使用 `src/` 和相关配置文件进行开发；本单文件版本用于确保演示环境稳定可打开。
