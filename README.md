# 团队项目执行看板

当前版本为**可离线打开**的单文件应用（`index.html`），并新增了你要求的可编辑能力。

## 已实现
- 首页总览（自动汇总）
- 任务清单（可新增/编辑/删除）
- 甘特排期（带日期表头）
- 成员负荷（可新增/编辑/删除成员，自动重算饱和度）

## 打开方式
### 方式 A（推荐）
```bash
python -m http.server 4173
```
访问：`http://127.0.0.1:4173`

### 方式 B
直接双击 `index.html`。

## 说明
- 本页不依赖 npm、不依赖外部 CDN，受限网络也可使用。
- 仓库中仍保留 React + TypeScript + Tailwind 的工程脚手架，便于后续切回工程化开发。
